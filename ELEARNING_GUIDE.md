## For Teachers: Using Scratch

Once you click "Enter Coding Classroom":

1. **You'll arrive at Scratch.mit.edu**
2. **No login needed** to explore projects
3. **To save projects:**
   - Click "Join Scratch" (free)
   - Create a teacher account
   - Set up a class
4. **Explore the "Explore" section** for inspiration
5. **Try the tutorials** to learn the basics
6. **Create your first project** by clicking "Create"

### Getting Started with Scratch:

- **Tutorials available** on the Scratch website
- **Step-by-step guides** for beginners
- **Example projects** to remix and learn from
- **Community support** and resources
